PlaybackSync — template adapter smoke test

This page exists to exercise the _template adapter. Open it with the ?pbsync-template query parameter — without that the adapter stays inert (as it would on any real site).

Expected with the extension loaded and ?pbsync-template set:

  • The page console logs [playbacksync:template] adapter activated.
  • Clicking play / pause and scrubbing the seek bar logs [playbacksync:bg] <tabId> { kind: "intent", ... } in the background service-worker console.
