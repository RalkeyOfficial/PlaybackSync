# Product Mission

## Problem

Friend groups who self-host Nextcloud cannot watch videos together without relying on external watch-party services. Existing solutions require either a high-bandwidth relay server (to stream video) or a third-party platform — neither of which fits a low-end self-hosted setup.

## Target Users

Small friend groups who self-host Nextcloud and want to watch video content together. The tool is designed for intimate, trusted groups — not large communities or organizations.

## Solution

PlaybackSync is a decentralized watch-party system built as a native Nextcloud app. It synchronizes playback state (play, pause, seek) across all participants without relaying video — the server only coordinates timing, so it works on low-end networks. Being Nextcloud-native means no external services are needed; everything runs within the users' own instance.
